# ZoomCue 🚀
### Studio-Quality Product Demos. Instant. Cinematic. Professional.

ZoomCue is the premier desktop companion for creators, developers, and product managers who need to create stunning video walkthroughs and demos without the high-end production costs. 

Designed for high-impact communication, ZoomCue transforms simple screen recordings into polished, cinematic experiences with intelligent auto-zooming, motion blur, and professional-grade editing tools.

## 📥 Getting Started
ZoomCue is built for performance on all major platforms. Download the latest release from the [Official Repository](https://github.com/TemitopeGX/zoomcue_app/releases) and start creating in seconds.

### **Windows**
1. Download the latest `.exe` installer.
2. Run the installer and launch ZoomCue from your dashboard.

### **macOS**
1. Download the latest `.dmg` file.
2. Drag ZoomCue to your Applications folder.
3. *Note:* Since ZoomCue is a high-performance system utility, you may need to grant **Screen Recording** and **Accessibility** permissions in System Settings.

### **Linux**
1. Download the `.AppImage` file.
2. Grant execution permissions: `chmod +x ZoomCue-Linux-*.AppImage`
3. Launch and enjoy professional recording on your favorite distro.
